JavaScript

JavaScript es el lenguaje de programación de HTML y de la web. JavaScript es facil de aprender. Es un lenguaje interpretado, por lo que no se necesita compilar el codigo, y todo se ejecuta del lado del cliente.

Los tags se definen entre los símbolos <tag>

Importar JavaScript

inline

Dentro del documento HTML

    console.log("Hola");

</script>

Importando el documentos js externo
