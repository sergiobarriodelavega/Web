JavaScript

Indice

JavaScript es el lenguaje de programación de HTML y de la web. JavaScript es facil de aprender. Es un lenguaje interpretado, por lo que no se necesita compilar el codigo, y todo se ejecuta del lado del cliente.

Los tags se definen entre los símbolos <tag>

Importar JavaScript

Existen dos formas de importar codigo de JavaScript

inline

Dentro del documento HTML

<script>

    console.log("Hola");

</script>

Importando el documentos js externo

<script src="miScript.js">

console.log()

Nos permite imprimir mensajes por la consola dev (F12)

alert()

Nos sirve para sacar por pantalla notificaciones

Variables

A diferencia de otros leguajes de programación, en JavaScript no se necesita especificar de que tipo es la variable

var miString = "Hola";
var miNumero = 1;

document.getElementBy...

Podemos seleciconar elementos del DOM según como lo queramos identificar

JS Descripción
getElementById("ID") Nos devuelve el objeto con el id especificado
getElementsByClassName("Clase") Nos devuelve un array de objetos que tengan la clase especificada
getElementsByTagName("Etiqueta") Nos devuelve un array de objetos que tengan la etiqueta especificada

Nos devuelve el elemento con el id especificado

document.getElementById("demo").innerHTML = "Hello JavaScript";

Editar y obtener propiedades

Podemos editar y obtener los valores de las propiedades de un elemento. Por ejemplo, para cojer el texto de un elemento usariamos:

document.getElementById("demo").innerHTML = "Ahora lo cambio por lo que quiera"
