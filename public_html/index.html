URL Encode

Las URL o Uniform Resource Locator, identifican en la web a un recurso al que el usuario quiere acceder.

La estructura común es:

scheme://prefix.domain:port/path/filename

Palabra Significado
scheme Define el tipo de servicio (normalmente http o https)
prefix Define el prefijo de dominio (Normalmente www)
domain El dominio de la web
port Define el número de puerto (por defecto es 80 "http")
path Define la ruta al servicio (Si no se identifica, sera el index de la web)
filename El nombre del fichero al que queremos acceder
