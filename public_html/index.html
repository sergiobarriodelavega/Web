HTML

Lenguaje de marcas que se utiliza para el desarrollo de páginas web. HTML hace referencia a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto.

Los tags se definen entre los símbolos <tag>

!DOCTYPE html

Indicamos que le fichero es de tipo HTML

<!-- Comentario -->

Nos sirve para crear comentarios. Los comentarios también aparecen en inspeccionar elemento de los navegadores, por lo que es importante cuidar los comentarios que ponemos.

html

Sirve para indicar que el fichero es de tipo html.

head

En él definiremos parametros de la pagina web, como codificación, título del a web etc.

title

Establecemos el título de la pestaña de nuestra página web.

body

Indicamos el comienzo del cuerpo de la página html. Este contiene todos los elementos visuales de la web.

h1 ... h6

Nos permite crear titulos. Indicaremos su nivel de importancia mediante números. Estos van del 1 al 6, siendo 1 el más importante, y del cual solo puede haber uno. A menor importancia, más pequeña es la letra.

a

Creamos enlaces. Tiene dos atributos:

  • href: Indicamos cual es el enlace al que se quiere ir
  • target: Indicamos si queremos que se abra el enlace en una nueva pestaña (_blank).

target

Podemos hacer diferentes target

_blank

Abrimos el enlace en una nueva pestaña

Enlace que se abre en una nueva pestaña

_self

Es el que viene por defecto. Indica que el enlace se abrira en la misma pestaña en la que estamos

Enlace que se carga en la misma pestaña

Podemos hacer referencia a enlaces externos e internos

_parent

Abrimos el enlace en un frame

Enlace que se abre en un frame

_top

Abrimos el enlace en el cuerpo de la ventana

Enlace que se abre en en el cuerpo de la ventana

Ir al CSS de enlaces

img

Insertamos imágenes. Dentro de la etiqueta img tenemos dos atributos. src, en el cual indicamos el origen de la imagen, y alt, en el que establecemos un texto que aparecerá en la imagen al poner el cursor encima. También tenemos los atributos width, y height, con los que establecemos el tamaño de la imagen


map

Define el mapa de una imágen. De esta forma definimos areas en las que se puede hacer click dentro de la imagen.

Información detallada en W3Schools

button

Listas

ul

Son listas sin ordenar, con símbolos

ol

Son listas ordenadas, con números ...

li

Es cada uno de los elementos de la lista.

Ejemplo:

  • Lunes
  • Martes
  • Miércoles
  • Jueves
  • Viernes
  • Sábado
  • Domingo

<ol type="A">

Podemos cambiar el tipo de numeracion en listas, definiendo el tipo de enumeración que queremos

Existen varios tipos:

  • type="1" La lista se enumera con numeros (Por defecto)
  • type="A" La lista se enumera con letras mayúsculas
  • type="a" La lista se enumera con letras minúsculas
  • type="I" La lista se enumera con numeros romanos en mayúsculas
  • type="i" La lista se enumera con numeros romanos en minúsculas

Tablas

Para definir tablas usamos la etiqueta <table>

Las tablas tienen los siguientes elementos:

  • Filas
  • Celdas
  • Cabezeras
  • Titulos (captions)

Filas

Se defininen con <tr>

Celdas

Se definen con <td> dentro del elemento de fila (tr)

Cabezeras

Dentro de las filas, podemos declarar cabezeras de la tabla, las cuales reciben un formato en negrita

colspan

Nos permite unir varias celdas de una tabla en la direccion de izquierda a derecha

rowspan

Nos permite unir varias celdas de una tabla en la direccion de arriba a abajo

Titulo/caption

Podemos ponerle titulos a las tablas con la etiqueta caption dentro del tag <table>

Ir a CSS de tablas

div

Tambien conocido como bloque o contenedor, nos permite agrupar elementos dentro de él.

Los divs cojen por defecto el 100% del ancho, mientras que mantienen al minimo la altura

span

Agurpa elementos pero a diferencia de div, no ocupa todo el ancho posible.

De esta forma podemos modificar texto dentro de otros elementos como un parrafo
