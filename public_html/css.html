CSS

Indice

CSS hace referencia a Cascading Style Sheets (Hojas de Estilo en Cascada)

CSS se puede añadir al html de 3 formas

  • Inline
  • Internal
  • External

Inline

Declaramos los estilos dentro de los elementos en HTML

<h1 style="color:lime;">Este texto se veria de color verde</h1>

Internal

Declaramos estilos dentor de la etiqueta style

  <style>
    #id {
      background-color: blue;
      border: 2px red solid;
    }
    .clase {
      font-width: bold;
      margin-bottom: 10px;
    }
    elemento {
      color: white;
      padding: 2px;
    }
  </style>

External

Declaramos todos los estilos en un documento css que enlazaremos al html con la etiqueta link dentro del <head>

<link rel="stylesheet" href="estilo.css">

Selectores de CSS

Nos permite aplicar estilos a dichos elementos

Tenemos 3 tipos:

id

Se identifica con #. Se usa para elementos unicos, es decir, solo puede existinr un id con el mismo nombre atribuido a un solo elemento

Se lo podemos agregar a un elemento añadiendole el atributo id="nombreId"

class

Se identifica con . . Se usa cuando queremos aplicar un mismo estilo a varios elementos.

Se lo podemos agregar a un elemento añadiendole el atributo class="nombreClase"

Nombre del elemento

Podmeos aplicar estilos a elementos del documento indicando el nombre del elemento (tag) en css

Ejemplo. Un estilo a todos los elementos p del documento:

    color: white;

Float

Sirve para posicionar y formatear contenido.

Puede tener los valores:

  • left: El elemento flota a la izquierda
  • right: El elemento flota a la derecha
  • none: El elemento no flota
  • inherit: Hereda el valor de su padre

Posiciones

Define la posición y como debe posicionarse

Posición Descripción
static El elemento se posiciona con normalidad, siguiendo la corriente por defecto del elemento
relative Se posiciona relativamente al la posicion por defecto (statica)
fixed El elemento se posiciona segun el punto de vista, por lo que el elemento se mantiene en el mismo sitio
absolute Se posiciona relativamente al elemento padre más cercano
sticky El elemento se posiciona relativamente al scroll del usuario

Mira los ejercicios a continuación en los que se aplican las diferentes posiciones
